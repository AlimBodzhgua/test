## Getting Started

Чтобы запустить проект, сначала необходимо скоипровать репозиторий:

```bash
git clone https://github.com/AlimBodzhgua/test.git
```

Затем нужно установит все необходимые зависимости:

```bash
npm install
```

Далее можно запустить dev server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере, чтобы увидеть результат.
