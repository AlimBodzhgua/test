## Getting Started

First, you need to clone this repository:

```bash
git clone https://github.com/AlimBodzhgua/test.git
```

Then you need to install all dependencies:

```bash
npm install
```

Now you can run this project on development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
